### Refactor decisions

- **`_BERRY_MARKERS` as a tuple of `(name, predicate)`** ‚Äî same shape
  as the other three Open/Closed seams. Predicate handles file-vs-dir
  asymmetry (`.yarn` is a directory; the rest are files). Could have
  hardcoded the predicate per entry inline; the tuple shape pays for
  itself the first time a Berry marker fork (Yarn 5+?) needs a third
  predicate kind (symlink? glob?).
- **Did not bump probe `version` attribute.** Schema `$id` carries the
  contract change. Probe `version` is a cache-key invariant pinned by
  existing S2-02 tests (`P.version == "0.1.0"`). The story didn't ask
  for it; surgical-changes principle.
- **Family-strip via `.split("-",1)[0]`** ‚Äî works because no
  package-manager identifier inside the enum contains a hyphen except
  `yarn-*`. If a future entry like `pnpm-workspaces` ever lands, this
  becomes wrong; an explicit map (`{"yarn-classic": "yarn",
  "yarn-berry": "yarn", "bun": "bun", ...}`) would be more robust.
  Deferred ‚Äî premature for one yarn split (Rule 2).

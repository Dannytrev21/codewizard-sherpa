### Mechanism note (AC-1 PR-body content)

`coverage.py`'s JSON report uses `percent_branches_covered` while the
synthetic test fixtures used the shorter `percent_covered_branch` alias.
The script tolerates both keys (real key preferred; falls back to alias)
so the AC-7 synthetic-dict inputs and the AC-13 real `coverage.json`
both feed `check()` without a translation layer.

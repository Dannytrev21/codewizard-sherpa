## Context

This is **the load-bearing story of the entire phase**. It transcribes `localv2.md §4`'s `Probe` ABC and four dataclasses byte-for-byte into `src/codegenie/probes/base.py`, then plants the snapshot test that pins the contract in CI. Once this lands, the ABC is frozen: every later phase (1–14) inherits the same `Probe` shape, and any drift — code-side or doc-side — fails CI with a message routing the offender to `templates/adr-amendment.md`. A transcription error encoded on day one becomes a permanent error inherited by every probe ever written. The regen script, not your retyping, is the source of truth.

Foundational work: nothing else in Step 2 (and almost nothing in Phase 1+) can compile without `probes/base.py`.

## Implementation outline

1. **File the `localv2.md §4` `Logger` amendment first.** §4 references `Logger` without importing it; per ADR-0007's resolution policy you cannot silently fix this in `base.py`. Open the doc-amendment PR (template at `templates/adr-amendment.md`, which you'll author below in step 7) that adds `from logging import Logger` to §4's import block — at this point in the story `templates/adr-amendment.md` does not yet exist, so author it inline as part of the same PR. The PR contains: (a) the §4 import-block edit, (b) `templates/adr-amendment.md`, (c) the rest of the steps below. The amendment workflow being exercised IS the deliverable.
2. Write `scripts/regen_probe_contract_snapshot.py` **second** — pure helpers `extract_section_4_body`, `normalize_and_hash`, `structural_signature` (signatures and contracts per AC-2 / AC-3). The extractor must anchor on `^## 4\. The probe contract\s*$` exactly and raise `ValueError` on zero or multiple matches. The normalizer is `re.sub(r"\s+", " ", body).strip()`; hashing uses `hashlib.sha256(normalized.encode("utf-8")).hexdigest()` (UTF-8 only, pin in test). `structural_signature(module)` walks the module's `dataclasses.fields()` and `inspect.get_annotations()` and serializes per the AC-3 schema (class-name-keyed alphabetical dict, field/method/class-attribute lists preserving Python 3.11+ insertion order, decorators captured). Add type hints and docstrings — tests import these as a library.
3. Run the regen script from the post-amendment `localv2.md §4`. Capture `doc_fingerprint` **before** writing `base.py`. This locks the "doc is source of truth" direction.
4. Implement `src/codegenie/probes/base.py` by literal copy from `localv2.md §4` (paste from the rendered doc, then diff line-by-line). Module contains, in this order: (a) module docstring + the `# TODO(S5-02): CODEOWNERS entry required …` line per AC-9; (b) imports exactly as §4 specifies (`from abc import ABC, abstractmethod`, `from dataclasses import dataclass`, `from typing import Literal, Any`, `from pathlib import Path`, `from logging import Logger` — the last only after the §4 amendment has merged); (c) four dataclasses `RepoSnapshot`, `Task`, `ProbeContext`, `ProbeOutput`; (d) `class Probe(ABC):` with class attributes `name`, `layer`, `tier`, `applies_to_tasks`, `applies_to_languages`, `requires`, `declared_inputs`, `timeout_seconds = 300`, `cache_strategy = "content"` (in this order) plus methods `applies()`, `cache_key()`, and `@abstractmethod async def run(self, repo: RepoSnapshot, ctx: ProbeContext) -> ProbeOutput`. No `version` attribute. Resolve any whitespace-only differences in favor of the doc; structural drift fails the regen.
5. Run the regen script again to populate `structural_signature`; commit `tests/snapshots/probe_contract.v1.json`.
6. Write `tests/unit/test_probe_contract.py` per the TDD plan below — anchoring tests, fixture-based extractor/normalizer tests, mutation-killer signature tests, failure-message-contract tests, stdlib-only-imports test. Import regen helpers from `scripts.regen_probe_contract_snapshot`; confirm `pyproject.toml`'s `[tool.pytest.ini_options].pythonpath = ["."]` (set in S1-02) makes the bare-script import work.
7. Write `templates/adr-amendment.md` with the five-step checklist per AC-6 (edit `localv2.md`, run regen, update `base.py` if structural drift surfaces, commit regenerated snapshot, reference ADR-0007 in PR).
8. Manually diff `src/codegenie/probes/base.py` against `localv2.md §4` (post-amendment) before opening the PR (per Implementation-level risk #1).

        pytest.fail(
            f"{SNAPSHOT_PATH} is missing. Regenerate with "

import hashlib


@pytest.mark.parametrize("body", [
    "a   b\tc\nd",
    "a   b\tc\r\nd",
    "  a   b   ",
    "a\n\n\n\nb",
])
def test_normalize_and_hash_collapses_whitespace_and_is_idempotent(body):
    h1 = normalize_and_hash(body)
    h2 = normalize_and_hash(" " + body + " \n")
    h3 = normalize_and_hash(body.replace("\n", "\r\n"))
    assert h1 == h2 == h3


def test_normalize_and_hash_is_utf8_sha256_of_collapsed_body():
    body = "  foo\t bar  "

    @dataclass
    class Inner:
        pass
    # build a tiny dataclass dynamically with a controllable field

        ns[field_name] = default
    Inner = dataclass(type("Inner", (), ns))
    m.Inner = Inner
    return m
